# Instructions for Claude Code

Two fixes to the Architecture tab. Both are aesthetic/presentation, no logic changes.

## Fix 1 — Drop "· FAILED" from Terminal state subtitle

**Problem:** After the AWAITING_HITL → AWAITING shortening in 16.1.2, the subtitle "DELIVERED · ARCHIVED · HELD · AWAITING · FAILED" still clips at the right edge of the SVG viewbox. The visible result is "DELIVERED · ARCHIVED · HELD · AWAITING · F..." — truncated.

**Fix:** drop "· FAILED" from the subtitle string so it reads "DELIVERED · ARCHIVED · HELD · AWAITING" and fits fully within the node width.

The detail panel (when a user hovers/clicks Terminal state) should still enumerate all five states including FAILED. Only the summary subtitle on the diagram itself gets shortened. FAILED remains a canonical terminal state in the architecture — it's just not shown on the diagram's summary strip.

## Fix 2 — Slow the entrance animation

**Target:** roughly 8-10 seconds total entrance, with each major beat given clear breathing room.

**Approach:** restructure `_runEntrance()` from a flat cascade (all nodes then all edges) to an interleaved sequence where each node appears, then its outgoing arrow draws, then the next node appears. This matches the natural reading order and gives each pipeline stage a moment to land.

**New timeline (implemented):**

| Stage | Appears at |
|-------|-----------|
| ingest node | 200ms |
| ingest→signalscribe arrow | 700ms |
| signalscribe node | 1200ms |
| signalscribe→post_ss arrow | 2000ms |
| post_ss node | 2400ms |
| post_ss→buatlas arrow | 2900ms |
| buatlas node (fan-out moment) | 3300ms |
| buatlas→pushpilot arrow | 4300ms |
| pushpilot node | 4700ms |
| pushpilot→pre_deliver arrow | 5400ms |

| edge labels fade in | 7900–8380ms |
| total | ~9.9s |

**Replay button:** shows "Replaying…" disabled state while animation runs; re-enables at 9.9s. Uses `.arch-replay-btn__label` span for text swapping.

**Reduced motion:** JS checks `prefers-reduced-motion: reduce` and collapses to 200ms instant fades with no stagger. CSS @media rule retained as belt-and-suspenders.

**Node transition:** 380ms → 650ms (cubic-bezier(0.2,0.8,0.2,1) preserved).
**Arrow draw duration:** ~400ms → 550ms base (cubic-bezier(0.4,0,0.2,1) preserved).
